# 💰 Finance Tracker (Full Stack)

## 🚀 Features
- User Authentication (JWT)
- Add Income & Expense
- Filter by Category & Date
- Dashboard Summary (Income, Expense, Balance)
- Chart Visualization

## 🛠 Tech Stack
- Backend: Node.js, Express
- Database: MongoDB
- Frontend: HTML, CSS, JavaScript
- Auth: JWT

## 📦 Project Structure
- finance-backend/
- finance-frontend/

## ▶️ Run Locally

### Backend
cd finance-backend  
npm install  
npm start  

### Frontend
Open index.html in browser
